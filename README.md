## Description du projet
Conception, déploiement et présentation d’une infrastructure réseau et système complète pour l’entreprise fictive **WorldSkills Lyon 2025 (WSL2025)** en collaboration avec **WorldSkills France (WSFR)** via une liaison MAN privée.

- 6 jours de travail + 1 jour démo + 1 jour soutenance  
- Équipements réels Cisco (1941, 2901, 3750, 2960) + VMs Proxmox VE & VMware ESXi  
- Routage OSPF + iBGP, VLANs, VTPv2, firewall (libre), VPN site-to-site & remote access, CA racine, Ansible automation  

**Le fonctionnel prime sur tout le reste !**
